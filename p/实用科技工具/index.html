实用科技工具

Oct 19, 2025

Docusaurus

Docusaurus是Facebook推出的开源框架，一些知名开源项目如React Native、KubeVela都采用它搭建官网。它可以让项目维护者专注于文档编写，无需在文档版本化、文档搜索、第三方集成等站点功能问题上花费精力。因为Docusaurus对这些都提供了内置支持，项目维护者只需要编写Markdown，提交PR，合并即可。

Tabby

Tabby是一个高度可定制的跨平台终端工具，支持Windows、MacOS和Linux三大操作系统。它自带SFTP功能，支持多种主题，有着现代化的UI界面和丰富的插件。例如saveoutput插件可以把控制台的输出保存成文件，syncconfig插件可以把Tabby的配置文件上传到GitHub进行云备份。

Bruno

Bruno是一款API测试工具，是Postman的开源轻量替代品，目前仅支持离线使用。其目的是为了保护数据隐私，确保数据安全性。Bruno的一些关键概念跟Postman一样，若之前用过Postman，使用Bruno几乎可以秒上手。它支持全类型的HTTP Method，还可以通过编写js代码设置请求前和请求后的逻辑，以及添加环境变量来快速完成测试。

dbwer

dbwer是一个免费的跨平台数据库工具，分为社区版和专业版，社区版开源免费，功能足够使用。它几乎支持所有的关系型数据库，用户可以右键创建连接，填写相关信息后即可连接数据库，还可以自动搜索驱动并下载。

Prompt Optimizer

Prompt Optimizer是一个AI提示词优化工具。它的部署方式多样，包括线上版本、自己部署、下载桌面应用以及使用Chrome插件等。用户可以通过它优化提示词，比如将一个普通的提示词进行系统式优化，使其符合要求，还可以对系统提示词进行优化，指定语言、描述风格背景、行文方式、规则和工作流程等。

MCP Toolbox

MCP Toolbox是Google开发的数据库管理工具。它可以通过MCP方便地控制数据库，具有使用简单（10行代码就能使用）、性能好、安全性高以及具有良好的观察性等特点。它支持多种数据库种类，通过配置管理模块注入到工具，再由执行代码把它转换成SQL语句，然后进入数据库连接。

Cloudia

Cloudia是Claude Code的UI界面。它可以完成基本的项目对话管理，对多个项目进行隔离；能保存历史数据；还可以通过CC agent根据不同项目需要定制代码提示模块。此外，它还能通过仪表盘跟踪使用情况以及模型的一些趋势。

BiliNote

BiliNote是一款开源的AI视频笔记助手，支持通过哔哩哔哩、YouTube等视频链接，自动提取内容并生成结构清晰、重点明确的Markdown格式笔记。它还支持本地音频转写，利用Fast - Whisper等本地音频转写工具提取语音内容，提升转写效率。

mad - professor

mad - professor是一款集PDF处理、AI翻译、RAG检索、AI问答及语音聊天于一体的学术论文阅读工具。导入PDF后它会自动提取、翻译和结构化论文内容，并且支持中英文对照阅读。它还支持AI智能问答，分屏界面，左侧论文右侧AI问答，可以语音提问和TTS语音回答。

CodeInterviewAssist

CodeInterviewAssist是一款开源的AI面试辅助工具，主要用于编程面试。它可以帮助用户分析面试题目、生成解决方案，甚至调试代码。它能自动分析截图的面试题目，提取关键信息生成详细的解决方案，包括代码和解释，还会分析时间和空间复杂度。如果代码有问题，可以实时调试，给出改进建议。

KoalaWiki

KoalaWiki是一个基于AI的代码知识库平台，可以自动解析Git代码仓库，生成比较详细、结构化的代码文档和项目见解。它支持添加和管理多个Git代码仓库，仓库分析完成后，可通过导航树浏览代码文档。该项目可用于新开发人员快速上手、遗留项目的知识梳理、技术文档标准化等。

deepwiki - open

deepwiki - open是一个代码说明书自动生成器。给它一个GitHub/GitLab地址，它就能自动分析代码结构和关系，自动生成该项目的详细说明文档，还能画出图表来解释代码是怎么运作的，支持自动生成Mermaid图表展示架构和数据流，支持Docker或本地部署。
